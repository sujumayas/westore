This is one of the most important item attributes. Please select one of the existing categories or create one. Every new category items will be sold on the market until you create a specific store for them.
The item Quality is going to be a number between -100 (pretty destroyed item) and +100 (Absolutely gorgeous perfect condition surrealisticly clean and new item). This will affect the shown price of the item. For example, if the item has +100 Quality, the store is going to try and sell it as expensive as it can. But, if the item is -100 Quality, they probably are going to sell it at normal default price. If the appraisal skill of the user who is buying is good, they can change this default store behaviors.
This is the Default Price. This is the price that should appear in the Book. When this item is shown to the users its going to be modified by their skill in Appraisal, the year they are playing and the special conditions you set when creating a Store. Actually, you can modify the final price for an item whenever you want, but this Default Price stays the same.
